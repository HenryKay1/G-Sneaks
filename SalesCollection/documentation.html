Description

  • At a high level, my component's make function takes in an object as input and builds a sortable product collection,integrated with an image slideshow functionality for each product in the collection, based on the input object's values
  • The makeSortableCollection has a pubic function, setImgDimensions to set image dimensions maybe based on your type of product.(This could be done via css but we need a public function and it specifies image tweaking to a specific collection
  • Our makeSortableCollection function's parameter object(paramObject) take in the following properties (paramObjects's (Key: Data Type) Syntax
    1.   productCollection: Array of Objects Array of Object's (Key: Data Type) Syntax Comment: (from JSON File, Database, hard-coded, etc ) [ { productId: Integer, productName: String, brand: String stock: Integer, wholesalePrice: Integer retailPrice: Integer Rating: Decimal or Double images: Object Object's (Key: Data Type) Syntax { color1: String, Comment: (color1's URL), color2: String, Comment: (color2's URL), .... } }, ... ]
    2.   title: String
    3.   sortPhrase: String
    4.   locale: String
    5.   currrency: String
    6.   NumColums: Integer Comment: (This is used to calculate the appropriate width for all products in collection, based on given margin)
    7.   HrzMarginSum: Integer Comment: (This is used to calculate the appropriate width for all products in collection, based on num of rows)
    8.   sortPropertyAliases:: Object Object's (Key: Data Type) Syntax Comment: (Passes in sort options and specifies the UI representation for that and the sort order (ascending or descending)) { propertyName1 : Array of Strings Array Syntax["Desired Name","Ascending or Descending"] Comment: (propertyName1 has to be an existing productCollection's existing array of Object's key) propertyName2: Array of Strings Array Syntax["Desired Name","Ascending or Descending"] Comment: (propertyName2 has to be an existing productCollection's existing array of Object's key) ..... }
    9.   newProductPropColl:: Array of Objects Array of Object's (Key: Data Type) Syntax Comment: (Defining a function and its parameters. The return value will be added to the productCollection's existing array of Object's keys) [ { prodPropName: String, funcParameters: Array of Strings Comment: (String Array consists of some number of productCollection's existing array of Object's keys) prodFunction: function(paramsArray) Comment: (paramsArray is the array of strings in funcParameters: , listed right above. Also,Function must return a text or a number }, ... ]
